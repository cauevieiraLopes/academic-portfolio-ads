# 📊 Sistema de Gestão de Notas e Estatísticas

Este projeto é uma atividade prática desenvolvida para a disciplina de **Lógica de Programação** no curso de ADS do **SENAI CIMATEC**.

## 📖 Descrição
O algoritmo foi desenvolvido em **C++** para automatizar o processo de lançamento de notas de uma turma. O sistema permite definir a quantidade de alunos, valida se as notas inseridas estão entre 0 e 10, calcula a média individual e, ao final, gera um relatório estatístico completo da turma.

## 🛠️ Tecnologias e Conceitos Aplicados
- **Linguagem:** C++
- **Conceitos de Lógica:**
  - **Validação de Dados:** Uso de estruturas `do-while` para garantir entradas corretas.
  - **Estruturas de Repetição:** Uso de `for` para processar múltiplos alunos dinamicamente.
  - **Acumuladores e Contadores:** Cálculo de média geral e contagem de aprovados/reprovados (Média de corte: 6.0).
  - **Lógica de Extremos:** Algoritmo para identificar a maior e a menor média entre todos os alunos processados.

## 🚀 Como Executar
1. Certifique-se de ter um compilador C++ (ex: g++) instalado.
2. Compile o arquivo:
   ```bash
   g++ main.cpp -o sistema_notas
3. Execute o programa:
  ```bash
  ./sistema_notas
  ```
